1) Добавил форму аутентификации на клиенте.
После запуска приложения выполняется попытка подключения к серверу, в случае успеха пользователю выводится
    окно аутентификации в модальном режиме. После успешного прохождения аутентификации в правой части диалогового окна
    отображается содержимое удаленной директории. Содержимое локальной директории всегда доступно в левой части окна.
Для взаимодействия графического интерфейса с сетевым подключением реализовал свой вариант паттерна Observer, добавив интерфейсы:
	public interface ConnectionObservable {
    		void registerObserver(ConnectionObserver observer);
    		void removeObserver(ConnectionObserver observer);
    		void notifyAllObserver();
    		void notifyObserver(ConnectionObserver observer);
	}

	public interface ConnectionObserver {
    		void updateConnectionState(ConnectionStatus connection, AuthStatus auth);
	}

2) Добавил функционал загрузки и выгрузки файлов из/в удаленное файловое хранилище.
Передача данных выполняется частями (buffer = 64Кб) в синхронном режиме:
	- Источник подготавливает файл для чтения и отправляет команду начала операции загрузки/выгрузки получателю
	- Получатель подготавливает файл для записи и отправляет команду источнику и готовности принимать данные
	- Источник считывает часть данных в буфер и отправляет получателю, если файл прочитан полностью отправляется
	    команда завершения операции загрузки/выгрузки
	- Получатель записывает переданный буфер в файл и отправляет сообщение с командой готовности принимать следующую часть
	- В случае если получатель принял команду завершения операции он закрыват файл и уведомляет источник об успешном
	    завершении операции
	- Все исключения ввода/вывода в файл отлавливаются и второй стороне отправляется уведомление о прекращении операции
	    загрузки/выгрузки, что позволяет корректно закрыть файл
Синхронный подход позволил:
 	- использовать один экземпляр буфера для загрузки и передачи файлов избежав лишнего копирования.
	- передавать несколько файлов одновременно, а также выполнять другие операции с хранилищем.
